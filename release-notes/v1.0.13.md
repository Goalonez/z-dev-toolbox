## 中文
- 优化格式转换工具的结构化输出行为：JSON 转 XML 时若缺少单一根节点会自动补上 `root`，JSON 对象现在也可以直接转换为单行 CSV、HTML Table 和 HTTP query 参数，同时补充了 query string 的 HTTP 解析兼容。
- 统一右上角全局提示的交互体验：将各工具的成功提示文案收敛为一致风格，缩短提示停留时间，并降低提示层的视觉遮挡感。

## English
- Improved structured output behavior in the format converter: JSON to XML now auto-wraps content with a `root` node when needed, plain JSON objects can now be converted directly to single-row CSV, HTML tables, and HTTP query strings, and HTTP parsing now also accepts query string input.
- Unified the top-right global toast experience: aligned success messages across tools, shortened the toast duration, and reduced the visual weight of the notification layer.
