## 中文
- 桌面端新增应用内更新检查与安装入口，可直接读取 GitHub Release 上的最新版本说明。
- 接入 Tauri updater 插件，并补齐桌面端 updater 构建、签名校验和 `latest.json` 生成链路。

## English
- Added in-app update check and install entry points for the desktop app, including GitHub Release notes in the UI.
- Integrated the Tauri updater plugin and completed the desktop updater build, signature validation, and `latest.json` manifest pipeline.
- Fixed the Tauri release config path in the GitHub Release workflow so desktop builds on all three platforms can load the updater config correctly, while still preventing web-only or extension-only releases from taking over the latest pointer.
