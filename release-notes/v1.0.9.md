## 中文
- 重构桌面端发布产物收集逻辑，改为使用跨平台 Node 脚本统一扫描并整理 macOS、Windows、Linux 的安装包、updater 产物与签名文件，去掉 workflow 中脆弱的 shell 路径与命名假设。
- 修复 macOS 发布阶段未能正确收集 `.app.tar.gz` updater 产物的问题，同时保留 Windows `bash` 执行修复，确保三平台桌面发布链路在同一版 workflow 下保持一致。

## English
- Reworked desktop release asset collection to use a cross-platform Node script that scans and prepares installer bundles, updater artifacts, and signatures for macOS, Windows, and Linux, removing the fragile shell-based path and filename assumptions from the workflow.
- Fixed the macOS release step so `.app.tar.gz` updater artifacts are collected correctly, while keeping the Windows `bash` shell fix in place to make the desktop release pipeline consistent across all three platforms.
