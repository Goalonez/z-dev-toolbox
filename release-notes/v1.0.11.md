## 中文
- 精简 GitHub Release 资产列表：Windows 和 Linux 不再额外上传一份与桌面安装包内容相同、仅名称不同的 updater 安装包，减少 release 中的重复产物。
- 调整 updater 清单生成逻辑，Windows 和 Linux 的 `latest.json` 直接引用对应桌面安装包，同时继续保留所需签名文件；macOS 仍保留独立的 `.app.tar.gz` updater 产物。

## English
- Reduced duplicate GitHub Release assets by no longer uploading an extra updater installer for Windows and Linux when it is identical to the desktop installer bundle apart from the filename.
- Updated updater manifest generation so `latest.json` points Windows and Linux entries to the corresponding desktop installers while still keeping the required signature files; macOS continues to ship its dedicated `.app.tar.gz` updater artifact.
