## 中文
- 首次正式发布 `Z Dev Toolbox 1.0.0`，提供 Web、浏览器插件和桌面端三种使用形态。
- 内置常用开发工具集，覆盖 JSON 格式化、格式转换、Base64、URL 编解码、时间转换、哈希、正则、二维码、文本 Diff、颜色转换、Crontab 预览和雪花 ID 生成。
- 统一基于共享核心能力和本地优先架构构建，偏好和使用状态保存在当前平台本地环境中。
- 发布流程已接入 GitHub Release、Docker Hub 和多平台桌面端资产产出。

## English
- First stable release of `Z Dev Toolbox 1.0.0`, available as a web app, browser extension, and desktop app.
- Ships with everyday developer utilities including JSON formatting, format conversion, Base64, URL encode/decode, time conversion, hashing, regex testing, QR code tools, text diff, color conversion, crontab preview, and Snowflake ID generation.
- Built on shared local-first core capabilities so preferences and usage state stay on the current platform.
- Release automation is wired for GitHub Releases, Docker Hub images, and multi-platform desktop artifacts.
