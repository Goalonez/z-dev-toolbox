## 中文
- 桌面端新增应用内更新检查与安装入口，可直接读取 GitHub Release 上的最新版本说明。
- 接入 Tauri updater 插件，并补齐桌面端 updater 构建、签名校验和 `latest.json` 生成链路。
- 修复 GitHub Release 工作流中的 Tauri 项目根目录、release 配置路径和 updater `pubkey` 注入逻辑，确保桌面端三平台发布能正确生成 updater 产物，并继续避免仅 Web 或插件渠道的 release 抢占 latest 指针。

## English
- Added in-app update check and install entry points for the desktop app, including GitHub Release notes in the UI.
- Integrated the Tauri updater plugin and completed the desktop updater build, signature validation, and `latest.json` manifest pipeline.
- Fixed the Tauri project root, release config path, and updater `pubkey` injection in the GitHub Release workflow so desktop releases on all three platforms can generate updater artifacts correctly, while still preventing web-only or extension-only releases from taking over the latest pointer.
