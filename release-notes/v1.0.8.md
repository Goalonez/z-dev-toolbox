## 中文
- 修复 GitHub Release 工作流在 Windows runner 上写入 Tauri release 配置时的 shell 兼容性问题，显式改为使用 `bash` 执行，避免发布阶段因默认 PowerShell 语法差异而失败。
- 保持桌面端 updater 构建链路不变，继续生成 Windows、macOS 和 Linux 所需的桌面安装包、updater 产物与签名文件。

## English
- Fixed the shell compatibility issue in the GitHub Release workflow when writing the Tauri release config on the Windows runner by explicitly using `bash`, preventing release failures caused by PowerShell syntax differences.
- Kept the desktop updater pipeline unchanged so Windows, macOS, and Linux releases still produce the installer bundles, updater artifacts, and signature files.
